## Reordering the menus

Enable **"Enable custom menu ordering"**, then give each menu group an order number in the fields below it:

- **Lower number = higher up**, higher number = lower down.
- **Blank or `0`** leaves a group in its default position. Negative numbers (e.g. `-5`) float a group to the very top.
- Groups are ordered relative to each other, so just nudge the ones you want to move.
- For the cleanest look, also turn on **"Hide all separators"** — separators stay in their original spots while items move around them.
- This overrides the default "Reorder options for tab context menu" ergonomics.

Example: to put the close options at the top of the tab menu and push "New Tab" to the bottom, set _[Tab] 'Close Tab / Undo Close' options — order_ to `-5` and _[Tab] 'New Tab' — order_ to `5`.
